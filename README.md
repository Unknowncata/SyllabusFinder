# シラバスファインダー chrome 拡張機能 (Syllabus Finder Chrome Extension)

シラバスファインダーは、青山学院大学学生のためだけに作られた Chrome 拡張機能です。ぜひお楽しみください。

## 搭載機能

-   シラバスの迅速な検索と簡単なアクセス
-   Tailwind CSS を使用した美しいデザイン
-   最新の Chrome 拡張機能 API (Side Panel) を活用

## 使い方

1. Chrome ツールバーより、シラバスファインダーのアイコンをクリック。
2. 学部・学科をリストから選択してください。（これは後で変更できないので注意してください。）
3. シラバスの検索が開始します。（数秒かかる場合があります。）（「CoursePower の講義一覧にアクセスしてください」というエラーが表示された場合は、CoursePower 講義一覧ページに移動し、再度お試しください。）
4. 読み込み終了後、検索したいクラスをクリックして、シラバスにアクセスしてください。

## インストール方法

-   Chrome ウェブストアからインストールする（推奨）: [Chrome ウェブストア](https://chrome.google.com/webstore/detail/syllabus-finder/your-extension-id) から拡張機能をインストールしてください。
-   または、リポジトリをクローンし、Chrome のデベロッパー モードを有効にしたうえ、「拡張機能」ページで「パッケージ化されていない拡張機能を読み込む」オプションを使用して開発目的で読み込むこともできます。

## 注意事項

-   シラバスファインダーが提供するシラバスの正確性を保証できませんので、提供された情報は各自必ずご確認ください。
-   一部の講義には利用可能なシラバスがない場合があります。そのような講義は灰色のフィルターで表示され、クリックできません。
-   英語版は近日公開予定ですが、まだ利用できません。申し訳ありません！ :(
-   今現在、自分の講義でのテストしかしていません！一部のユーザーには正しく動作しない可能性が高いです。 もし、問題が発生した場合は、GitHub リポジトリで Issue を立ててください。 [GitHub リポジトリ](https://github.com/Unknowncata/SyllabusFinder/)
